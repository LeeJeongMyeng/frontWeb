[1단계:개념] 6. 하이프 링크의 target 옵션을 기술하세요.


하이퍼링크의 target 속성 target속성에 따라 새로운 창, 계층 구조의 부모 창, 특정한 frame변경 등을 link로 처리 할 수 있다.

하이퍼링크 타겟 종류

  • _blank : 새로운 윈도우 창 로딩하면서 페이지 출력
  • _self : 현재 윈도우 창에서 변경(default)
  • _parent : 부모 윈도우 창을 변경
  • _top : 브라우저 윈도를 변경 처리
  • frame01 : 지정해놓은 프레임을 가져옴
